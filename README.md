Analyzing the US Car Market (2022–2023) — Power BI Dashboard
This project is a comprehensive Power BI dashboard built to analyze US car sales (2022–2023) using a Kaggle dataset. The goal was to transform raw sales data into clear, actionable insights through data cleaning, data modeling, and advanced DAX measures.

Key features

Sales Overview: KPIs for total sales, average price, and vehicles sold.
Trend Analysis: YTD and weekly trends with company-wise performance comparisons.
Structural Breakdown: Sales distribution and performance by body style.
Granular Detail: An interactive table for row-level exploration and deeper analysis.

Skills & tools used: Power Query (data preparation), data modeling (relationships/star schema), DAX (measures/KPIs), and interactive Power BI visuals.
