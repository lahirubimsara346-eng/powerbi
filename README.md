Analyzing the US Car Market (2022–2023) — Power BI Dashboard

Developed a comprehensive Power BI dashboard to analyze US car sales data (2022–2023) using a Kaggle dataset. The project focused on transforming raw sales data into actionable business insights through data cleaning, modeling, and advanced analytics.

Key Features

Sales Overview: KPIs for total sales revenue, average vehicle price, and units sold.
Trend Analysis: Year-to-date (YTD) and weekly sales trends with manufacturer-wise performance comparisons.
Market Structure Analysis: Sales distribution and performance by vehicle body style.
Interactive Exploration: Drill-down capabilities and detailed tables for granular analysis.

Tools & Techniques

Power BI
Power Query (Data Cleaning & Transformation)
Data Modeling (Star Schema & Relationships)
DAX (Measures, KPIs, and Time Intelligence)
Interactive Data Visualization
